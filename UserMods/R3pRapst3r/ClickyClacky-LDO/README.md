# ClickyClacky Buffer for LDO Crossbar 

This is a simple modification of the existing ClickyClacky buffer for using openbuilds angle brackets with the LDO crossbar.

Please note, there is currently only support for 300mm and 350mm builds with standard length docks.

Openbuilds angle brackets are used to mount the crossbar to the frame the same way the bed extrusions do.
- See pages 18-19 of the [Voron 2.4r2 assembly manual](https://github.com/VoronDesign/Voron-2/blob/Voron2.4/Manual/Assembly_Manual_2.4r2.pdf).

## Parts to print:
Everything from `Clicky-Klack > Standard docks > (300 or 350)` folder in this repo, except `Left.stl` and `Right.stl`

Enable supports for the modified parts.
### 300mm build
- 1x Standard-300-Left.stl
- 1x Standard-300-Right.stl
### 350mm build
- 1x Standard-350-Left.stl
- 1x Standard-350-Right.stl

## BOM:
- 2x [openbuilds 2020 angle brackets](https://kb-3d.com/store/frame-structure/56-2020-2-way-corner-extrusion-connector-black-openbuilds-1534037881565.html)
- 2x M5 1mm shim
- 2x M5x10 BHCS
- 2x M5 T-nuts
